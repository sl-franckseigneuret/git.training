# Récupération du projet

## durée: 2 min

Depuis C:\Projects :

`git clone https://github.com/USERNAME/git-training.git`

Le projet est alors cloné sur votre machine.

`ls` pour lister les répertoires présents et `cd git-training` pour atteindre le répertoire.

`git remote -v` et noter que le (fetch) et le (push) sont rattachés à votre repository et non au projet original.
On pourra updater le fetch pour récupérer les derniers updates du projet.

Il est aussi possible d'utiliser l'explorateur pour naviguer dans le projet
