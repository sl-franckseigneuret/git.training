# Avant de démarrer

## Avec un navigateur

1. Créer un compte git : <https://github.com/>
2. Aller sur : <https://github.com/yterraillon/git-training>
3. Cliquer sur le bouton "Fork" en haut à droite
4. Aller sur : <https://github.com/YOURUSERNAME?tab=repositories>
5. S'assurer que le repository est bien présent

## Localement

1. Définir un répertoire de travail : `C:\Projects`
2. S'assurer que git et git Bash sont bien installés.
3. Démarrer Git Bash dans le répertoire de travail.
